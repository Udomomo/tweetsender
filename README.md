# tweetsender
tweet-only Twitter CLI

## When to use
- When you want to tweet while coding
- But don't want to be distracted by your friend's funny posts

## Installation

## Usage
- `tweet {your tweet}`
    - send tweet

- `tweet -l {your tweet}`
    - check your tweet's length (this doesn't send your tweet)

- `tweet -s`
    - set your Twitter API Token

- `tweet -h`
    - show help
